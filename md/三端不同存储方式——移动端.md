## 推荐你这样做

你的 AI 聊天项目可以用这个方案：

```
1. SQLite 存聊天记录
   - conversations
   - messages
   - memories
   - characters

2. Capacitor Filesystem 存文件
   - Live2D 模型
   - 语音文件
   - 图片
   - 导出的聊天记录

3. unstorage / IndexedDB 存轻量配置
   - 当前选择的角色
   - UI 设置
   - 临时缓存
```

最像微信/QQ的核心方案是：

> **聊天文本和索引用 SQLite，图片/音频/模型文件用 App 沙盒文件系统。**

  记忆系统特性

  - 四种类型: user, feedback, project, reference
  - 自动提取: session 切换时 LLM 自动提取值得记忆的信息
  - 自动注入: 每次聊天时自动将记忆注入到系统提示中
  - 限制规则: 200 行 / 25KB，超出时回退到索引模式
  - 新鲜度提示: 根据更新时间自动添加"昨天更新/X天前更新"提示
